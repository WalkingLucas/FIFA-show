# 修改日志

## 0.2.1 - 2026-06-17

### 调整

- 直播比赛新增进球监听：检测到新增进球后立即刷新比分数据。
- 单场详情会同步 ESPN summary 返回的最新比分、状态和比赛时钟，避免详情页比分滞后。

## 0.2.0 - 2026-06-17

### 新增

- 支持点击直播和 FT 比赛进入单场详情页。
- 单场详情接入 ESPN summary 数据，显示进球、助攻、牌、换人和文字直播摘要。
- 直播详情页每 60 秒自动刷新，右键刷新可立即更新当前详情。
- 详情页支持滚轮查看完整事件列表。
- 新增单场详情缓存，刷新失败时继续显示上一次结果。

### 调整

- 收藏球队不再改变比分页、历史页和赛程页的比赛排序。
- README 补充数据源、刷新策略、详情页和版本说明。

## 0.1.0 - 2026-06-17

### 新增

- Windows 桌面世界杯比分悬浮窗初版。
- 无边框、半透明、永远置顶、不显示在任务栏。
- 接入 ESPN 世界杯赛程和比分数据。
- 支持正在进行、最近结束和接下来比赛展示。
- 支持球队收藏、历史/未来翻页、紧凑模式和右键菜单。
- 支持赛前预测和进球、助攻、纪律、趣味榜单。
